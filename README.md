## вңҰ What is Enclave?

Imagine opening your phone and stepping into a **personal universe** вҖ” your own space, your own apps, your own rules. That's Enclave.

Enclave is a **Virtual Space Application** for Android. It acts as a beautiful mini-launcher that lives *inside* a single app. No root required. No system modifications. Just pure, fluid experience.

You get:
- рҹҢҢ A stunning animated virtual space as your home
- рҹ“ұ Mini-apps living inside the space as floating cards
- рҹ“Ң One-tap shortcut pinning to your Android home screen
- вҡЎ 60fps animations. Always. No compromises.

---

## вңҰ Why Enclave Exists

Most phones are cluttered. Apps everywhere. No personality. No flow.

Enclave gives you **a space that is entirely yours** вҖ” organized, beautiful, and blazing fast. Think of it like having a premium private dashboard inside your phone. Everything you need, nothing you don't, wrapped in a UI that actually feels good to use every single day.

## вңҰ Animation Philosophy

Enclave is built on one rule: **every interaction must feel alive.**

We use **spring physics** вҖ” not just easing curves вҖ” so animations feel physical and natural, never robotic.

```kotlin
// Every icon tap uses this вҖ” feels like pressing a real button
spring(
    dampingRatio = Spring.DampingRatioMediumBouncy,
    stiffness = Spring.StiffnessLow
)
```

Performance targets:

- рҹҺҜ **60fps** minimum on all supported devices
- рҹҡҖ **120fps** on high refresh rate displays
- рҹӘ¶ **Zero jank** вҖ” all animations run on GPU via `graphicsLayer`

## вңҰ Performance Rules

These rules are enforced throughout the entire codebase:

1. **Vector Drawables only** вҖ” no PNG icons, ever
2. **`graphicsLayer` for all animations** вҖ” GPU-accelerated, zero jank
3. **`key()` in every list item** вҖ” prevents unnecessary recomposition
4. **`derivedStateOf` for computed values** вҖ” no redundant calculations
5. **Low-RAM device detection** вҖ” static fallback for older phones
6. **R8 full mode in release** вҖ” smallest possible APK size
